    How to Use

    1. 填参数，或者直接点预设。
    2. 点击 Generate stata.lic。
    3. 如果程序要求输入安装参数，就填写上面的 Serial number、Code、Authorization。
    4. 如果程序直接读取许可证文件，就复制或下载下面生成的 stata.lic。

    Linux: 当前本地安装的稳定入口路径是：

    ~/.local/opt/stata-mp/runtime/stata.lic
    • Windows / macOS: 如果安装器或初始化窗口要求填表，优先直接填写上面的三项。
    • 如果目标副本已经存在 stata.lic，也可以直接替换那份文件。
    • MP 用 stata-mp 或 xstata-mp 启动；BE 保持 field7 为空。
